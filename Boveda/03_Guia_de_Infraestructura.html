La VCard ha pasado de ser un render HTML simple, a una Arquitectura completa "Desacoplada" (Headless Front-end comunicándose con Google Apps Script).

Estructura del Proyecto Entregable

Tu proyecto cuenta con los siguientes componentes principales compilados y minificados para máximo rendimiento:

fiore-de-enero/ # Proyecto Raíz Desplegado
index.html # Core principal público
style.css # Estilos con CSS variables Dark/Gold
script.js # Motor de UX e iterador del catálogo
admin/ # Panel Privado (NO SE DEBE RENOMBRAR)
index.html # Dashboard administrativo UI (PIN GATE SHA-256)
app.js # Controlador fetch PUT/POST para sheets (Token)
vercel.json # Security Headers (Clickjacking, XSS)
README_DATABASE_SETUP.md # Backend Source (Script)

Infraestructura de Seguridad (Fortaleza Treze)

  • Security Headers en Edge: Vercel inyecta automáticamente cabeceras como X-Frame-Options: SAMEORIGIN y X-XSS-Protection para evitar que bots clonen tu sitio o lo incrusten en otras páginas (Clickjacking).
  • Despacho Seguro (Webhook Token): Solo tu panel administrativo posee la llave secreta para escribir en Firebase/Google Sheets. Si los hackers intentan enviar peticiones POST directas, serán rechazados.
  • Criptografía en Cliente: El código PIN (`2026`) está hasheado criptográficamente (SHA-256) en el frontend. Si se inspecciona el código, solo se verá el hash, nunca la contraseña en texto plano.

Hosting, Costos Cero y Arquitectura Severless

  • Frontend CDN: Desplegado mediante Github y la arquitectura Global Edge de Vercel. (Sin mantenimiento de Hosting, SSL forzado, HSTS, 99.9% up-time).
  • Base de Datos y Webhook (Backend): El catálogo, descripciones de platillos y configuración es orquestada en Google Apps Script a través de Google Sheets, la conexión se realiza gracias a una petición HTTP(S) async-fetch no-cors. Coste y límite mensual de despliegues en servidores back-end convencionales eludidos estratégicamente.
  • Alojamiento Multimedia: Gráficos estáticos integrados; banners de promoción interactivos o Canva iframes optimizados.
El panel administrativo en la carpeta /admin permite editar productos en la VCard. La Bóveda Google Sheet funge solo como receptor.
